Nested Pairs

The Grammar

The grammar.bnf provides a demonstration of nested recursion. It is a grammar for matching nested brackets with text. The nesting allows for multiple nested brackets within any bracket pair.

In this example, the brackets have been chosen as angle brackets, <>, but any bracket pair could have been used. The only requirement is that the pattern (bracket) for the rule L canâ€™t match the pattern for the rule R.

The Input

The file bracket-text.txt is the input. It has a somewhat arbitrary selection of nested brackets with text between the brackets.
